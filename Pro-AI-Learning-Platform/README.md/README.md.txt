# Pro AI Learning Platform 🎓🤖

This project is developed for **NEXATHON 1.0**.  
It is an AI-powered student learning platform with login, subjects, quiz, motivation, and AI tutor features.  

## 🚀 Features
- 🔐 Student Login System  
- 📚 Subjects with learning resources  
- 📝 Daily Quiz (random 5 questions)  
- 🤖 AI Tutor (Ask your questions using OpenAI)  
- 💡 Daily Motivation for students  
- 👤 Student Profile Dashboard  
- 📞 Helpline Section  

## 🛠️ Tech Stack
- Python  
- Streamlit  
- OpenAI API
